Adds **UNDERTALE** battles (and more!) to Kristal!

## This library adds...
* UNDERTALE battles (obviously) in the light (and optionally dark) world!
* All of UNDERTALE's weapons and armors!
* Functionality for Spells and TP!
* A Froggit encounter!
* ...And (hopefully) more!
